В Java наследование осуществляется с использованием ключевого слова extends. Это ключевое слово позволяет классу наследовать свойства и методы другого класса. Пример:

java
public class РодительскийКласс {
    // Поля и методы родительского класса
}

public class ДочернийКласс extends РодительскийКласс {
    // Дополнительные поля и методы дочернего класса
}

В этом примере ДочернийКласс наследует свойства и методы из РодительскогоКласса. Он может использовать их напрямую и/или переопределить, при необходимости.

Также важно помнить про ключевое слово super, которое используется внутри дочернего класса для обращения к членам родительского класса.

Пример переопределения метода:

java
public class РодительскийКласс {
    public void метод() {
        System.out.println("Метод из родительского класса");
    }
}

public class ДочернийКласс extends РодительскийКласс {
    @Override
    public void метод() {
        super.метод(); // Вызов метода из родительского класса
        System.out.println("Метод из дочернего класса");
    }
}

В этом примере метод метод() переопределён в дочернем классе, и при вызове он сначала вызывает метод родительского класса с помощью super.метод(), а затем добавляет свою логику.

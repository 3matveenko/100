В Java класс, объект и интерфейс - это основные строительные блоки объектно-ориентированного программирования (ООП).

  1. Класс (Class):

    • Определение: Класс - это шаблон или чертёж, описывающий состояние и поведение объектов. Он является абстракцией, которая объединяет данные (поля) и методы, обрабатывающие эти данные.
    • Пример:
      java
      public class Person {
    // Поля (данные)
    String name;
    int age;

    // Метод (поведение)
    public void sayHello() {
        System.out.println("Привет, меня зовут " + name);
    }
}

  2. Объект (Object):

    • Определение: Объект - это экземпляр класса. Он представляет конкретный элемент, созданный на основе класса, и обладает своим собственным состоянием и поведением.
    • Пример:
      java
      public class Main {
    public static void main(String[] args) {
        // Создание объекта класса Person
        Person person1 = new Person();
        person1.name = "Иван";
        person1.age = 25;

        // Вызов метода объекта
        person1.sayHello();
    }
}

  3. Интерфейс (Interface):

    • Определение: Интерфейс определяет набор методов, которые класс должен реализовать, но не предоставляет их конкретной реализации. Интерфейсы позволяют достичь множественного наследования методов в Java.
    • Пример:
      java
      public interface Shape {
    double getArea(); // Метод, который должны реализовать классы
}

public class Circle implements Shape {
    double radius;

    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
}

Надеюсь, это помогло вам лучше понять основные концепции класса, объекта и интерфейса в Java.

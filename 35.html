Класс POJO (Plain Old Java Object) - это простой Java-объект, который не зависит от каких-либо особенностей фреймворков или библиотек. Он представляет собой обычный объект, содержащий поля (переменные-члены) и методы доступа к этим полям (геттеры и сеттеры), но не имеющий дополнительных зависимостей или особенностей.

Пример класса POJO на языке Java:

java
public class Person {
    private String name;
    private int age;

    // Конструктор
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // Геттеры и сеттеры
    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }

    // Дополнительные методы, если необходимо
}

Этот класс представляет объект "Person" с полями "name" и "age", а также соответствующими геттерами и сеттерами. Такой класс легко создавать, поддерживать и использовать в различных частях программы.

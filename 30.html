В Java полиморфизм реализуется через следующие элементы языка:

  1. Перегрузка методов (method overloading): Возможность создания нескольких методов с одним и тем же именем в классе, но с разными параметрами. Компилятор выбирает подходящий метод на основе типов переданных аргументов.

    Пример:

    java
    public class MyClass {
    public int add(int a, int b) {
        return a + b;
    }

    public double add(double a, double b) {
        return a + b;
    }
}

  2. Переопределение методов (method overriding): Возможность предоставить новую реализацию метода в подклассе, который уже был определен в его суперклассе. Это позволяет объектам использовать методы своих суперклассов, но с различной реализацией.

    Пример:

    java
    public class Animal {
    public void makeSound() {
        System.out.println("Animal makes a sound");
    }
}

public class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("Dog barks");
    }
}

  3. Интерфейсы (interfaces): Интерфейсы в Java предоставляют способ реализации полиморфизма. Классы могут реализовывать интерфейсы, определяя свои собственные реализации методов интерфейса.

    Пример:

    java
    interface Shape {
    void draw();
}

class Circle implements Shape {
    @Override
    public void draw() {
        System.out.println("Drawing a circle");
    }
}

class Square implements Shape {
    @Override
    public void draw() {
        System.out.println("Drawing a square");
    }
}

Эти элементы языка вместе обеспечивают поддержку полиморфизма в Java.
